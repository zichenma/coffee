Cookie部分功能包含：$.setCookie()、$.getCookie()、$.delCookie()

$.setCookie()

作用：设置cookie。

参数：options包括name: cookie的名称、value:cookie 值、expires: 生存日期、path:有效路径、domain:域名、secure:是否加密。

返回值：返回一个指向当前元素的对象。

示例：
输入cookie的有关信息，可设置、获取、删除cookie; 

					  $("test1").bind("click",function(){
					     $.setCookie({
					           name: $("name").val(), // cookie名字
					           value: $("value").val(), //cookie内容
					           expires: $("day").val(), //生存日期
					           path: "/", //有效路径
					           domain: "", // 指定域名
					           secure: false //是否加密
					       });
					  });
					  
					  $("test2").bind("click",function(){ 
					          $("demo1").html($.getCookie($("name").val()));//获取cookie  
 					          $("demo1").style("background-color","red"); 
					  });
					  $("test3").bind("click",function(){ 
					          $.delectCookie($("name").val());//获取cookie 
					          $("demo1").html(" "));// 
					           
					  });
