$().bind(type, handler)

作用：为当前元素绑定事件处理。

参数：type为事件类型，如”click“，”mouseover"，“blur”等，handler事件触发时执行的操作

返回值：返回一个指向当前元素的对象。

示例：
为方块1和按钮绑定单击事件。单击方块1或者单击按钮，可触发事件。 

					    var i=0;
					    $("demo1").bind("click",function(){ //为demo1绑定单击事件操作。
					       $("demo1").html("共单击了"+(++i)+"次~");
					    });
					    $("test1").bind("click",function(){//为run1绑定单击事件。
					       $("demo1").html("单击了div~");
					    });

getButton

作用：检查鼠标的那个健被点击

参数：无。

返回值：返回被按下的鼠标键值。 1是中间的滚轮，2是右键，0为左键。

示例：
判断鼠标的那个健被点击了？ 

					
					 $("test6").bind("click",function(e) {
							var event = markyun.Event.getEvent(e);//得到统一的event
							var button = markyun.Event.getButton(event);//鼠标那个健被点击了？
							var bt;
							if (button === 0) {
								bt = "鼠标左键";
							} else if (button === 2) {
								bt = "鼠标右键"
							} else if (button === 1) {
								bt = "鼠标滚轮";
							} 
							$("demo7").html("第" + (++i) + "次按下~:然后" + bt + "被点击了!");
						});

$().getMousePos(event)

作用：获取当前鼠标的坐标位置。

参数：event

返回值：返回一个长度为2的数组，分别为水平X和垂直Y坐标。

示例：
获取当前鼠标的位置 

					
					 $().bind("mousemove",function(event){ 
					       var m=$().getMousePos(event);
					       $("demo4").html("当前鼠标的(X位置:"+m[0]+",y位置:"+m[1]+")");
					 });

.getCharCode(event)

作用：取得Keypress事件时，获取按下键的ASCII编码。

参数：event

返回值：返回一个按下键的ASCII编码。

示例：
获取当前鼠标的位置 

					
					$("demo5").bind("keypress", function(e) {

						var keyCode = markyun.Event.getCharCode(event);//按下那个健的ASCII编码
						$("demo6").html("按键是内容是：" + keyCode) 
					});
