Base基础部分功能包含：$(selector)、 $().ready()、 $().html()、 $.extend()

$(selector)

作用：用于获取元素，并生成一个指向该元素的对象。

参数：selector为选择器的名字，默认值为document，目前支持元素对象和id，待完善的选择功能。

返回值：返回一个指向当前元素的对象。

示例：
选择id为test11的元素，并为其加上边框。 

						$("test1").bind("click", function(e) {
							  $("test11").style("border","2px solid #000"); 
						});

ready(function)

作用：监听DOM加载完成，通过 $(document).ready() 可以加载多个函数。

参数：参数为DOM加载完成后要运行的函数。

返回值：返回一个指向当前元素的对象。

示例：
在页面加载完成后，为元素添加样式。 

						
						$("demo2").style("background","#ccc");
DIV的样式是在页面加载后才添加的

$(selector).html(val)

作用：获取或设置当前元素节点的文本内容。

参数：val为设置的值，如果val为空，则获取当前元素节点的文本内容，否则，把当前元素节点的文本内容设置为val。

当val为空时，返回当前元素节点的文本内容；val存在时，返回一个指向当前元素的对象。

示例：
点击测试，设置id为demo3的文本内容。 

						
							$("test2").bind("click", function() {
								$("demo3").html("喜歡骑行、摄影、行为心理学;关注互聯網最新動态。");
							});
默认内容，您可以点击测试改变此div内容

$.extend(obj,prototype)

作用：扩展markyun对象本身。用来在markyun命名空间上增加新函数。

参数：obj用以扩展的对象，prototype为扩展的函数集合，如果参数只有prototype，则扩展新函数到markyun.js对象上，成为静态方法。方便直接调用

返回值：返回一个markyun对象。

示例：
选择运行时,为markyun.js对象添加两个函数，并运行。 

						
				 $("test3").bind("click",function(){  
				        
				        // 给markyun对象添加的功能（静态方法），可以直接点方法名进行操作
				        $.extend({
				        	trim : function(str) {
				    			return str.replace(/^\s+|\s+$/g, '');
				    		},
				    		now: function() {
				    			var today= new Date().toLocaleString();
				    			return today;
				    		}
				        });	
				        
				        // 扩展markyun.js对象。用来在fn命名空间上增加功能。例：
					    $.extend(markyun.prototype,{  
							// 移除事件绑定
							unbind : function(type, handler) {
								markyun.Event.removeEvent(this[0], type, handler);
								return this;
							},						
						});
						
				        alert($.trim("     给markyun对象添加的功能，去掉前后空格后的字符串        ")+"当前时间："+$.now());//使用新增的函数，获得返回值.
				    });

$.noConflict()

作用：交还对 $ 符号的使用权，方便其他(如：Jquery)使用。

参数：无

返回值：返回一个markyun对象。

示例：
选择运行时,为markyun.js对象添加两个函数，并运行,然后再次测试前面的demo将不能正确执行。 

						
						  $.noConflict();
